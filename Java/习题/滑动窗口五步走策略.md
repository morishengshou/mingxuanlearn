# 吴师兄滑动窗口“五步走”策略 
1. 定义维护变量
2. 定义窗口边界：定义窗口的起始位置 start 和结束位置 end，通常初始化为 0
3. 移动右端点：右端点 end 从 0 开始向右遍历数组，直到移动到尾部。

此时一般涉及更新需要维护的变量, 有的变量需要一个 if 语句来维护 (比如最大最小长度)
4. 更新维护变量/收缩左端点：
    1. 新增元素进入窗口，更新维护变量。
    2. 如果窗口内元素不满足条件（如哈希表中有重复元素），则将窗口左边的元素移出窗口，即 start 右移。

5. 记录结果：在移动过程中，根据需要记录满足条件的窗口最大长度、最小长度或对应的数值。
