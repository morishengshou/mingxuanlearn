# 配套背诵版总结

## Java Stream API 背诵口诀

**思想：**
声明式、内部迭代、惰性求值、一次性、少副作用

**结构：**
数据源 → 中间操作 → 终止操作

**高频：**
`filter map flatMap distinct sorted collect`

**结论：**
**擅长数据加工，不替代所有 `for`**

---

## LeetCode Stream 背诵口诀

**会用的地方：**
去重、排序、统计、分组、打平、判断

**别乱用的地方：**
双指针、滑窗、DP、回溯、图搜索

**总原则：**
**前处理后处理用 Stream，核心算法用 `for`**
